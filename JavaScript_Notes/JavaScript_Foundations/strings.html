Step 1: What is a String?

A string is text data. In JavaScript, strings are written using:

  • Double quotes "Hello"
  • Single quotes 'Hello'
  • Backticks (template literals) `Hello`
let name = "John";
let city = 'London';
let message = `Welcome to JavaScript`;

console.log(name);
console.log(city);
console.log(message);

Step 2: String Concatenation

Concatenation means joining strings together. 

let firstName = "Jane";
let lastName = "Doe";

let fullName = firstName + " " + lastName;
console.log(fullName); // Jane Doe

Using Template Literals (Recommended)

let age = 25;
let intro = `My name is ${firstName} and I am ${age} years old.`;

console.log(intro);

Template literals use backticks and ${} for inserting variables.

Mini Challenge: Combine your name and favorite hobby into one sentence using template literals.

Step 3: String Properties & Length

Strings have built-in properties. 

let text = "JavaScript";

console.log(text.length); // 10

Remember: string indexing starts at 0. 

console.log(text[0]); // J
console.log(text[4]); // S
Mini Challenge: Print the first and last character of a string.

Step 4: Common String Methods

JavaScript provides powerful built-in string methods. 

let str = "  Hello World  ";

console.log(str.toUpperCase());   // HELLO WORLD
console.log(str.toLowerCase());   // hello world
console.log(str.trim());          // removes spaces
console.log(str.includes("World")); // true
console.log(str.indexOf("World"));  // 8
console.log(str.slice(2, 7));       // llo W
console.log(str.replace("World", "JS")); // Hello JS

Strings are immutable — methods return a new string.

Step 5: Escape Sequences

Escape sequences allow special characters inside strings. 

let quote = "He said, \"JavaScript is awesome!\"";
let newLine = "First Line\nSecond Line";
let tabSpace = "Name:\tJohn";

console.log(quote);
console.log(newLine);
console.log(tabSpace);

Common Escape Characters:

  • \" → Double quote
  • \' → Single quote
  • \n → New line
  • \t → Tab
  • \\ → Backslash
Mini Challenge: Write a string that prints a two-line poem using \n.

Step 6: String Splitting & Joining

You can convert strings into arrays and back. 

let sentence = "JavaScript is powerful";

let words = sentence.split(" ");
console.log(words); // ["JavaScript", "is", "powerful"]

let joined = words.join("-");
console.log(joined); // JavaScript-is-powerful
Mini Challenge: Split a sentence into words and count how many words it contains.

Step 7: String Comparison

Strings can be compared using comparison operators. 

console.log("apple" === "apple"); // true
console.log("apple" === "Apple"); // false
console.log("b" > "a"); // true

String comparison is based on Unicode values.

Step 8: Mental Model – Strings in JavaScript

Think of strings as:

  • Text containers
  • Indexed sequences (like arrays of characters)
  • Immutable (cannot be changed directly)
  • Powerful tools using built-in methods

You can:

  • Create them
  • Combine them
  • Extract parts
  • Search inside them
  • Replace content
  • Transform case
