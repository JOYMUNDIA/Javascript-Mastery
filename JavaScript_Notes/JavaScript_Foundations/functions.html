From Zero Knowledge → Confident JavaScript Developer

Step 1: What Are Functions?

A function is a reusable block of code that performs a task. Instead of writing the same code multiple times, you can put it inside a function and call it whenever needed. 

function greet() {
  console.log("Hello, world!");
}

greet(); // Calls the function

Think of a function as a machine: you press the button (call it), and it does something for you.

Mini Challenge: Write a function that prints your name and call it.

Step 2: Function Parameters

Parameters are inputs you give to a function so it can work with different values. 

function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice");
greet("Bob");

- Here name is a parameter.
- When we call the function, we provide an argument like "Alice".
- This makes the function dynamic and reusable.

Mini Challenge: Write a function that takes two numbers and prints their sum.

Step 3: Return Values

Functions can return a value so you can use it later in your code. 

function add(a, b) {
  return a + b;
}

let result = add(5, 3);
console.log(result); // 8

- return gives the result back.
- Without return, the function just performs an action but doesn’t give anything back.

Mini Challenge: Create a function that multiplies two numbers and returns the result.

Step 4: Function Expressions

Functions can also be stored in variables. These are called function expressions. 

const greet = function(name) {
  console.log("Hello, " + name + "!");
};

greet("Charlie");

- Functions are first-class citizens in JavaScript — they can live in variables, arrays, or even objects.

Mini Challenge: Store a function that subtracts two numbers in a variable and call it.

Step 5: Arrow Functions

Arrow functions are a shorter syntax for writing functions. 

const add = (a, b) => a + b;

console.log(add(4, 7)); // 11

- If there’s only one expression, it automatically returns the result.
- You can also write multi-line arrow functions with braces and return.

Mini Challenge: Write an arrow function that squares a number.

Step 6: Default Parameters

You can give default values to parameters if no argument is passed. 

function greet(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greet();       // Hello, Guest!
greet("Alice"); // Hello, Alice!
Mini Challenge: Create a function that raises a number to a power, defaulting the exponent to 2.

Step 7: Functions Inside Functions

Functions can call other functions or even be nested. 

function square(x) {
  return x * x;
}

function sumOfSquares(a, b) {
  return square(a) + square(b);
}

console.log(sumOfSquares(2, 3)); // 13

- Nested or called functions make your code modular and easier to maintain.

Mini Challenge: Write a function that calculates the area of a rectangle using a helper function to calculate perimeter.

Step 8: Mental Model – Functions as Machines

Visualize functions as 3D cubes in your code world: - **Input ports** → parameters - **Engine** → the code inside - **Output port** → return value Every time you call the function, the cube processes the inputs and gives you outputs.
