Step 1: Rounding — Not All Rounding Is Equal

Math.round()
4.5 → 5
Math.floor()
4.9 → 4
Math.ceil()
4.1 → 5
---

1. Math.round()

Rounds to the nearest integer. 

Math.round(4.4) // 4
Math.round(4.5) // 5

Rule:

  • .5 goes UP
---

2. Math.floor()

Always rounds DOWN. 

Math.floor(4.9) // 4
Math.floor(-4.1) // -5 (important!)
---

3. Math.ceil()

Always rounds UP. 

Math.ceil(4.1) // 5
Math.ceil(-4.9) // -4
---

Real World Thinking

  • floor: safe minimum (e.g., money you definitely have)
  • ceil: required minimum (e.g., people needed)
  • round: general approximation

Step 5: Random Numbers (VERY IMPORTANT)

Core Idea

Math.random()

Returns: 

0 ≤ number < 1
Example: 
0.234234
0.912312
---

⚠️ This is NOT useful directly

You must transform it. ---

Random Integer Formula

Math.floor(Math.random() * max)
---

Example: 0 → 9

Math.floor(Math.random() * 10)
---

Custom Range (MOST IMPORTANT)

min + Math.floor(Math.random() * (max - min + 1))
---

Example: 50 → 100

let num = 50 + Math.floor(Math.random() * 51);
---

Mental Model

  • Math.random → gives base randomness
  • * multiply → scale range
  • floor → remove decimals
  • + min → shift range

Step 6: Number Formatting (CRITICAL SKILL)

This is where beginners stop… …and professionals begin. ---

1. toFixed()

let price = 12.3456;

price.toFixed(2) // "12.35"
⚠️ Returns a STRING! ---

Fix Floating Errors

(0.1 + 0.2).toFixed(2) // "0.30"
---

2. Converting Back to Number

Number((0.1 + 0.2).toFixed(2))
---

3. toPrecision()

Controls total digits: 
let num = 123.456;

num.toPrecision(4) // "123.5"
---

Step 7: Professional Number Formatting (REAL WORLD)

Intl.NumberFormat

This is what real apps use. ---

Currency Example

let formatter = new Intl.NumberFormat("en-US", {
  style: "currency",
  currency: "USD"
});

formatter.format(1234.5);
// $1,234.50
---

Local Currency (Important for real apps)

let zmw = new Intl.NumberFormat("en-ZM", {
  style: "currency",
  currency: "ZMW"
});

zmw.format(1500);
// ZK 1,500.00
---

Adding Commas

(1000000).toLocaleString()
// 1,000,000
---

Custom Decimal Places

new Intl.NumberFormat("en-US", {
  minimumFractionDigits: 2,
  maximumFractionDigits: 2
}).format(12.3)
---

Step 8: Real World Example

let price = 19.99;
let tax = 0.16;

let total = price + (price * tax);

let formatted = new Intl.NumberFormat("en-US", {
