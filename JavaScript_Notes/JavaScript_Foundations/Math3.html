Step 1: Clamping Values (VERY IMPORTANT)

Clamping means:

“Force a number to stay within a range.”
---

Example Problem

A game health system:

  • Minimum health = 0
  • Maximum health = 100
---

Solution

function clamp(value, min, max){
  return Math.max(min, Math.min(max, value));
}
---

How it Works

clamp(120, 0, 100) → 100
clamp(-10, 0, 100) → 0
clamp(50, 0, 100) → 50
---

Step 2: Normalization

Normalization converts values into a **0 → 1 scale**.

---

Formula

(value - min) / (max - min)
---

Example

let progress = (75 - 0) / (100 - 0); // 0.75
---

Use Cases

  • Progress bars
  • Sliders
  • Animations
---

Step 3: Interpolation (LERP)

Interpolation means:

“Move smoothly between two values.”
---

Formula

start + (end - start) * t
---

Example

let value = 0 + (100 - 0) * 0.5; // 50
---

Use Cases

  • Animations
  • Camera movement
  • Game physics
---

Step 4: Dealing with Precision (CRITICAL IN FINANCE)

Never trust floating point for money.

---

Problem

0.1 + 0.2 !== 0.3
---

Professional Solution

Use integers: 
let price = 1999; // cents

let tax = 160; // cents

let total = price + tax;

console.log(total / 100);
---

Why This Works

  • No floating point errors
  • Exact calculations
---

Step 5: Safe Calculations

Check Before Math

function safeDivide(a, b){
  if(b === 0) return null;
  return a / b;
}
---

Prevent NaN

function safeNumber(value){
  return Number.isFinite(value) ? value : 0;
}
---

Step 6: Random Systems (Game Level)

Weighted Random

Not all outcomes should be equal.

---

Example

function weightedRandom(){
  let rand = Math.random();

  if(rand < 0.7) return "common";
  if(rand < 0.9) return "rare";
  return "legendary";
}
---

Use Cases

  • Loot systems
  • AI behavior
  • Probabilities
---

Step 7: Rounding Strategy (REAL BUSINESS LOGIC)

Different industries use different rounding rules.

---

Banker's Rounding (Concept)

Sometimes .5 rounds to nearest EVEN number.

---

Custom Rounding

function roundTo(value, decimals){
  let factor = 10 ** decimals;
  return Math.round(value * factor) / factor;
}
---

Step 8: Performance Considerations

Math is fast, but repeated calculations matter.

---

Avoid Recomputing

❌ Bad: 
for(let i=0;i<1000;i++){
  Math.sqrt(144);
}
✅ Good: 
let root = Math.sqrt(144);

for(let i=0;i<1000;i++){
  root;
}
---
